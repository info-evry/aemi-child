## Description

Aemi Child is designed to be an aesthetic minimalist WordPress theme. It provides a highly legible, clean, clear and really minimalist theme. Aemi Child is also a really fast and efficient theme. Aemi Child is Gutenberg-ready.



## Installation

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Aemi Child in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install' button then click on the 'Activate' button to use your new theme right away.
